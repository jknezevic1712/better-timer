# Better Timer

## Running locally

### Production environment

1. `pnpm install`
2. `pnpm build`
3. `pnpm start`

### Development environment

1. `pnpm install`
2. `pnpm dev`

## Docker

### Using Docker compose

1. Run `docker compose up`

## Vercel

1. [App is also deployed on Vercel](https://better-timer.vercel.app)
